Future

In the future, I'd like to work as a software developer on either the backend or frontend (or maybe even both) of applications. I took the AWS Industry Sub-Major at UTS to hopefully give me some additional insight into backend development and cloud computing.

I also hope to learn some new technologies:

  • All the C's
  • Unity/GODOT
  • Rust
  • At least 500 more JavaScript frameworks (and their respective meta frameworks)
  • React Native — to create fast multi-platform apps with Expo
  • Electron — to create slow bloated apps with React
  • Flutter — so I can consider not learning the above two
  • Query languages like postgres and SQL

You'll notice I included some game engines, as I hope to make at least one game (preferrably in the horror genre) for fun.

I am also interested in creating a Google TV replacement for Triviaverse, a Netflix TV game which they cruelly wiped off the app for some reason (as if doubling the price for an ad-free plan wasn't enough).
